# Shopping Paradise Nursery

## Project Name
**Paradise Nursery Shopping Application**

## Description
Paradise Nursery is an e-commerce web application built with React and Redux for browsing and purchasing indoor plants. The application features a modern, user-friendly interface with a comprehensive shopping cart system.

## Features
- Browse indoor plants categorized by type
- Add plants to shopping cart
- Manage cart items (increase/decrease quantity, remove items)
- Real-time cart count display
- Responsive design
- Redux state management

## Technologies Used
- React.js
- Redux Toolkit
- React Router
- CSS3
- JavaScript (ES6+)

## Categories
- Air Purifying Plants
- Low Maintenance Plants
- Flowering Plants

## Installation

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate to project directory
cd paradise-nursery

# Install dependencies
npm install

# Start the development server
npm start
```
